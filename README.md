# Analisi Qualità dell'Aria a Milano

Questa applicazione Streamlit permette di visualizzare e analizzare i dati storici della qualità dell'aria a Milano (2016-2025). L'app elabora automaticamente i dati geografici e le misurazioni annuali per fornire grafici interattivi e mappe delle stazioni.

## Installazione e Avvio

### 1. Requisiti
Assicurati di avere **Python 3.8+** installato sul tuo sistema.

### 2. Installazione delle dipendenze
Apri il terminale nella cartella del progetto ed esegui il seguente comando per installare tutte le librerie necessarie:

```bash
pip install -r requirements.txt
```

### 3. Avvio dell'applicazione
Apri il terminale nella cartella del progetto ed esegui il seguente comando per avviare l'applicazione:
```bash
streamlit run app.py
```
Se il comando precedente non funziona, prova ad eseguire quest'altro:
```bash
python -m streamlit run app.py
```
